constructor

  • new MultiDictionary(toStrFunction?: function, valuesEqualsFunction?: IEqualsFunction<V>, allowDuplicateValues?: boolean): MultiDictionary
    • Defined in lib/collections.ts:1195

    Creates an empty multi dictionary.

    class

    A multi dictionary is a special kind of dictionary that holds multiple values against each key. Setting a value into the dictionary will add the value to an array at that key. Getting a key will return an array, holding all the values set to that key. You can configure to allow duplicates in the values. This implementation accepts any kind of objects as keys.

    If the keys are custom objects a function which converts keys to strings must be provided. Example:

    function petToString(pet) {
 return pet.name;
}

    If the values are custom objects a function to check equality between values must be provided. Example:

    function petsAreEqualByAge(pet1,pet2) {
 return pet1.age===pet2.age;
}
    constructor

    Parameters

    • Optional toStrFunction: function

      optional function to convert keys to strings. If the keys aren't strings or if toString() is not appropriate, a custom function which receives a key and returns a unique string must be provided.

        • Parameters

          Returns string

    • Optional valuesEqualsFunction: IEqualsFunction<V>

      optional function to check if two values are equal.

    • Default value allowDuplicateValues: boolean = false
